# /add-ui-kit — Component Showcase

Genera la biblioteca visual de componentes del proyecto — una ruta `/ui` que corre en local
y muestra todos los componentes con los tokens exactos del proyecto (colores, tipografía, radios).

Elimina la inconsistencia visual entre páginas. Una vez generado, los agentes deben
referenciar `/ui` antes de crear cualquier componente nuevo.

**Dos modos automáticos:**
- `FRESH` — proyecto nuevo: entrevista de branding → tokens → showcase
- `REDESIGN` — proyecto existente: scan → reporte de inconsistencias → estandarización
