# /add-payments — Sistema de Pagos (Polar o Stripe)

Integra pagos con decision tree para elegir el proveedor correcto:
- **Polar**: Merchant of Record — ideal para indie hackers sin empresa
- **Stripe**: Directo — para empresas con billing custom

Pre-requisito: `/add-login` debe estar ejecutado.

## Ejecucion

Leer y ejecutar `.claude/skills/add-payments/SKILL.md`.
El skill guia una entrevista para elegir proveedor antes de implementar.
